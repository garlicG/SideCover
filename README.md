SideCover
======================

概要　
----------
SideCoverはコンテンツの上側に表示されるカバーのようなウィジェットです。
 CoverはLeftSideかRightSideを指定します。
スワイプ操作でCoverを開け閉めすることもできます。  
Styleである程度のCoverの属性を指定できます。  
background,coverWidth,shadowWidh,shadowColor,透過アニメーションの有無
